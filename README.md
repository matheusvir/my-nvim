# Este é apenas um repositório para minha config pessoal do nvim

## Requisitos
- Sistema unix (Linux ou MacOS), neovim >= 0.11.0

## Como instalar
Basta executar:

```bash
cp -r ./nvim/ ~/.config/
```
E, posteriormente, executar o neovim.

## Obs:
- Para uma melhor experiência, instale uma **Nerd Font**.
- Deixei aqui um script para instalar a **JetBrains Mono** como Nerd Font.
- Para outras fontes, consulte: [Nerdfonts](https://github.com/ryanoasis/nerd-fonts).
